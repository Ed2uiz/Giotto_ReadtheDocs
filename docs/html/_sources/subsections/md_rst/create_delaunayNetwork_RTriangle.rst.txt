``create_delaunayNetwork_RTriangle``
========================================

create_delaunayNetwork_RTriangle

Description
-----------

Create a spatial Delaunay network with RTriangle

Usage
-----

.. code-block:: r

   create_delaunayNetwork_RTriangle(
     spatial_locations,
     sdimx = "sdimx",
     sdimy = "sdimy",
     Y = TRUE,
     j = TRUE,
     S = 0,
     ...
   )
