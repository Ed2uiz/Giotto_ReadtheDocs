
``get_giottoImage_MG``
==========================

Get magick -based giotto ``image``

Description
-----------

Get a giottoImage from a giotto object

Usage
-----

.. code-block:: r

   get_giottoImage_MG(gobject = NULL, name = NULL)

Arguments
---------

.. list-table::
   :header-rows: 1

   * - Argument
     - Description
   * - ``gobject``
     - giotto object
   * - ``name``
     - name of giottoImage `\ ``showGiottoImageNames`` <#showgiottoimagenames>`_


Value
-----

a giottoImage
