
``reconnect_giottoLargeImage``
==================================

reconnect_giottoLargeImage

Description
-----------

reconnect giottoLargeImage if image pointer is broken

Usage
-----

.. code-block:: r

   reconnect_giottoLargeImage(giottoLargeImage, image_path)

Arguments
---------

.. list-table::
   :header-rows: 1

   * - Argument
     - Description
   * - ``giottoLargeImage``
     - giottoLargeImage to reconnect
   * - ``image_path``
     - path to source file of giottoLargeImage


Value
-----

reconnected giottoLargeImage
