create_delaunayNetwork_RTriangle
--------------------------------

Usage
~~~~~

::

   create_delaunayNetwork_RTriangle(
     spatial_locations,
     sdimx = "sdimx",
     sdimy = "sdimy",
     Y = TRUE,
     j = TRUE,
     S = 0,
     ...
   )
