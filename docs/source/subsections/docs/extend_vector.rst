extend_vector
-------------

Description
~~~~~~~~~~~

extend the range of a vector by a given ratio

Usage
~~~~~

::

   extend_vector(x, extend_ratio)

Arguments
~~~~~~~~~

+-----------------------------------+-----------------------------------+
| ``x``                             | x                                 |
+-----------------------------------+-----------------------------------+
| ``extend_ratio``                  | extend_ratio                      |
+-----------------------------------+-----------------------------------+
