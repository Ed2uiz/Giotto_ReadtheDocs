Description
~~~~~~~~~~~

Converts .gef file (output stereo-seq pipeline) into transcript with
coordinates

Usage
~~~~~

::

   getGEFtxCoords(gef_file, bin_size = "bin100")

Arguments
~~~~~~~~~

+-----------------------------------+-----------------------------------+
| ``gef_file``                      | path to .gef file                 |
+-----------------------------------+-----------------------------------+
| ``bin_size``                      | bin size to select from .gef file |
+-----------------------------------+-----------------------------------+
