Get *magick*-based giotto ``image``
-----------------------------------

Description
~~~~~~~~~~~

Get a giottoImage from a giotto object

Usage
~~~~~

::

   get_giottoImage_MG(gobject = NULL, name = NULL)

Arguments
~~~~~~~~~

+-----------------------------------+-----------------------------------+
| ``gobject``                       | giotto object                     |
+-----------------------------------+-----------------------------------+
| ``name``                          | name of giottoImage               |
|                                   | ``showGiottoImageNames``          |
+-----------------------------------+-----------------------------------+

Value
~~~~~

a giottoImage
