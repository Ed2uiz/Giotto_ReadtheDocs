Get *terra*-based giotto ``largeImage``
---------------------------------------

Description
~~~~~~~~~~~

Set a giottoLargeImage from a giottoObject

Usage
~~~~~

::

   get_giottoLargeImage(gobject = NULL, name = NULL)

Arguments
~~~~~~~~~

+-----------------------------------+-----------------------------------+
| ``gobject``                       | giotto object                     |
+-----------------------------------+-----------------------------------+
| ``name``                          | name of giottoLargeImage          |
|                                   | ``showGiottoImageNames``          |
+-----------------------------------+-----------------------------------+

Value
~~~~~

a giottoLargeImage
