get_img_corners
---------------

Description
~~~~~~~~~~~

finds four corner spatial coords of giottoImages or magick-images

Usage
~~~~~

::

   get_img_corners(img_object)

Arguments
~~~~~~~~~

+-----------------------------------+-----------------------------------+
| ``img_object``                    | giottoImage or magick-image to    |
|                                   | use                               |
+-----------------------------------+-----------------------------------+
