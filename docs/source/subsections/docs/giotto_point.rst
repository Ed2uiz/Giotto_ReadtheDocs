giotto_point
------------

Description
~~~~~~~~~~~

scattermore::geom_scattermost

Usage
~~~~~

::

   giotto_point(plot_method = c("ggplot", "scattermore", "scattermost"), ...)

Arguments
~~~~~~~~~

+-----------------------------------+-----------------------------------+
| ``...``                           | geom_point parameters             |
