List cell ID names
------------------

Description
~~~~~~~~~~~

lists the available cell id names. In effect, these names are the
spat_units and poly info in the gobject

Usage
~~~~~

::

   list_cell_id_names(gobject)

Arguments
~~~~~~~~~

+-----------------------------------+-----------------------------------+
| ``gobject``                       | giotto object                     |
+-----------------------------------+-----------------------------------+

Value
~~~~~

vector with names of available sets of cell_IDs
