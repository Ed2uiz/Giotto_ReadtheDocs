Test if list element exists
---------------------------

Description
~~~~~~~~~~~

Test if nth element of list exists

Usage
~~~~~

::

   list_element_exists(x, index)

Arguments
~~~~~~~~~

+-----------------------------------+-----------------------------------+
| ``x``                             | list                              |
+-----------------------------------+-----------------------------------+
| ``index``                         | element index                     |
+-----------------------------------+-----------------------------------+

Value
~~~~~

boolean
