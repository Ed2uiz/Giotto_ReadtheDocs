List feat ID names
------------------

Description
~~~~~~~~~~~

lists the available feat id names In effect, these names are the
feat_types and feature info in the gobject

Usage
~~~~~

::

   list_feat_id_names(gobject)

Arguments
~~~~~~~~~

+-----------------------------------+-----------------------------------+
| ``gobject``                       | giotto object                     |
+-----------------------------------+-----------------------------------+

Value
~~~~~

vector with names of available sets of feat_IDs
