nnDT_to_kNN
-----------

Description
~~~~~~~~~~~

Convert a nearest network data.table to a kNN object

Usage
~~~~~

::

   nnDT_to_kNN(nnDT)

Arguments
~~~~~~~~~

+-----------------------------------+-----------------------------------+
| ``nnDT``                          | nearest neighbor network in       |
|                                   | data.table format                 |
+-----------------------------------+-----------------------------------+

Value
~~~~~

kNN object
