Description
~~~~~~~~~~~

Prints the available giotto images that are attached to the Giotto
object

Usage
~~~~~

::

   showGiottoImageNames(gobject)

Arguments
~~~~~~~~~

+-----------------------------------+-----------------------------------+
| ``gobject``                       | a giotto object                   |
+-----------------------------------+-----------------------------------+

Value
~~~~~

prints names of available giotto image objects
