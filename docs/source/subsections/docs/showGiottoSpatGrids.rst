Description
~~~~~~~~~~~

Prints the available spatial grids that are attached to the Giotto
object

Usage
~~~~~

::

   showGiottoSpatGrids(gobject, nrows = 4)

Arguments
~~~~~~~~~

+-----------------------------------+-----------------------------------+
| ``gobject``                       | giotto object                     |
+-----------------------------------+-----------------------------------+
| ``nrows``                         | number of rows to print           |
+-----------------------------------+-----------------------------------+

Value
~~~~~

prints name of available spatial grids
