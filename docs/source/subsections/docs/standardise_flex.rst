standardise_flex
----------------

~~~~~~~~~~~

::

Arguments
~~~~~~~~~

+-----------------------------------+-----------------------------------+
| ``x``                             | matrix                            |
+-----------------------------------+-----------------------------------+
| ``center``                        | center data                       |
+-----------------------------------+-----------------------------------+
| ``scale``                         | scale data                        |
+-----------------------------------+-----------------------------------+

Value
~~~~~

standardized matrix
