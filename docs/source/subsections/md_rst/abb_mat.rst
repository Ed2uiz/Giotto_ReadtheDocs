``abb_mat``
===========

Print abbreviated matrix

Description
-----------

print abbreviated matrix exprObj. Works for Matrix pkg denseMatrix,
matrix, data.frame and classes that inherit them.

Usage
-----

.. code:: r

   abb_mat(exprObj, nrows, ncols, header = TRUE)
