===============
addGiottoPoints
===============

:Date: 2022-09-22

Description
===========

Adds Giotto points to an existing Giotto object

Usage
=====

.. code:: r

   addGiottoPoints(gobject, gpoints)

Arguments
=========

+-------------------------------+--------------------------------------+
| Argument                      | Description                          |
+===============================+======================================+
| ``gobject``                   | giotto object                        |
+-------------------------------+--------------------------------------+
| ``gpoints``                   | list of giotto point objects, see    |
|                               | ```createGio                         |
|                               | ttoPoints`` <#creategiottopoints>`__ |
+-------------------------------+--------------------------------------+

Value
=====

giotto object
