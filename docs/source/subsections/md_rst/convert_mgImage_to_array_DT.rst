convert_mgImage_to_array_DT
---------------------------

Description
~~~~~~~~~~~

converts a magick image object to a data.table

Usage
~~~~~

::

   convert_mgImage_to_array_DT(mg_object)

Arguments
~~~~~~~~~

+-----------------------------------+-----------------------------------+
| ``mg_object``                     | magick image or Giotto image      |
|                                   | object                            |
+-----------------------------------+-----------------------------------+

Value
~~~~~

data.table with image pixel information
