``degrees``
===========

Radian/degree conversions

Description
-----------

Convert radians to degrees and vice versa

Usage
-----

.. code:: r

   radians(deg)
   degrees(rad)

Arguments
---------

======== ===========
Argument Description
======== ===========
``deg``  degrees
``rad``  radians
