=================
getDistinctColors
=================

:Date: 2022-10-06

Description
===========

Returns a number of distint colors based on the RGB scale

Usage
=====

.. code:: r

   getDistinctColors(n)

Arguments
=========

======== =======================
Argument Description
======== =======================
``n``    number of colors wanted
======== =======================

Value
=====

number of distinct colors
