==============
giottoToSeurat
==============

:Date: 2022-09-20

``giottoToSeurat``
==================

Convert Giotto to Seurat

Description
-----------

Converts Giotto object into a Seurat object

Usage
-----

.. code:: r

   giottoToSeurat(obj_use = NULL, ...)

Arguments
---------

+-------------------------------+--------------------------------------+
| Argument                      | Description                          |
+===============================+======================================+
| ``obj_use``                   | Giotto object                        |
+-------------------------------+--------------------------------------+
| ``...``                       | additional params to pass to         |
|                               | ```get_spatial_l                     |
|                               | ocations`` <#getspatiallocations>`__ |
+-------------------------------+--------------------------------------+

Value
-----

Seurat object
