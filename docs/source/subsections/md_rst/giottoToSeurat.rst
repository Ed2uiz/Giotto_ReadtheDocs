Description
~~~~~~~~~~~

Converts Giotto object into a Seurat object

Usage
~~~~~

::

   giottoToSeurat(obj_use = NULL, ...)

Arguments
~~~~~~~~~

+-----------------------------------+-----------------------------------+
| ``obj_use``                       | Giotto object                     |
+-----------------------------------+-----------------------------------+
| ``...``                           | additional params to pass to      |
|                                   | ``get_spatial_locations``         |
+-----------------------------------+-----------------------------------+

Value
~~~~~

Seurat object
