
``list_feature_info_names``
===============================

list_feature_info_names

Description
-----------

return the available names for giotto feature information

Usage
-----

.. code-block:: r

   list_feature_info_names(gobject)

Arguments
---------

.. list-table::
   :header-rows: 1

   * - Argument
     - Description
   * - ``gobject``
     - giotto object


Value
-----

vector with names of available feature information
