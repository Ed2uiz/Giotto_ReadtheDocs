
``list_spatial_info``
=========================

list_spatial_info

Description
-----------

return the available giotto spatial polygon information

Usage
-----

.. code-block:: r

   list_spatial_info(gobject)

Arguments
---------

.. list-table::
   :header-rows: 1

   * - Argument
     - Description
   * - ``gobject``
     - giotto object


Value
-----

names of available spatial polygon information
