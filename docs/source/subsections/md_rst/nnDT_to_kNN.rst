===========
nnDT_to_kNN
===========

:Date: 2022-09-22

Description
===========

Convert a nearest network data.table to a kNN object

Usage
=====

.. code:: r

   nnDT_to_kNN(nnDT)

Arguments
=========

======== =============================================
Argument Description
======== =============================================
``nnDT`` nearest neighbor network in data.table format
======== =============================================

Value
=====

kNN object
