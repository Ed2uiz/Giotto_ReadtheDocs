``objName-generic``
===================

Giotto object name information

Description
-----------

access and set name slot fo S4 subobject

Usage
-----

.. code:: r

   objName(x)
   list(list("objName"), list("nameData"))(x)
   list(list("objName"), list("nameData"))(x) <- value

Arguments
---------

========= ==========================================
Argument  Description
========= ==========================================
``x``     a Giotto S4 class subobject with name data
``value`` value to set as object name
