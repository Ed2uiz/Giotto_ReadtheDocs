
``scale_spatial_locations``
===============================

scale_spatial_locations

Description
-----------

Scale the X and Y coordinates given by the scale factor input

Usage
-----

.. code-block:: r

   scale_spatial_locations(spatlocs, scale_factor)

Arguments
---------

.. list-table::
   :header-rows: 1

   * - Argument
     - Description
   * - ``spatlocs``
     - spatial locations to scale
   * - ``scale_factor``
     - scaling factor to apply to coordinates
