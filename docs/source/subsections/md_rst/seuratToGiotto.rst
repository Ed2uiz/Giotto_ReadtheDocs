
``seuratToGiotto``
======================

seuratToGiotto

Description
-----------

Converts Seurat object into a Giotto object

Usage
-----

.. code-block:: r

   seuratToGiotto(sobject)

Arguments
---------

.. list-table::
   :header-rows: 1

   * - Argument
     - Description
   * - ``sobject``
     - Seurat object


Value
-----

Giotto object
