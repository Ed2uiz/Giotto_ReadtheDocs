==================
seuratToGiotto_OLD
==================

:Date: 2022-09-22

Description
===========

Converts Seurat object into a Giotto object. Deprecated, see
```giottoToSeurat`` <#giottotoseurat>`__

Usage
=====

.. code:: r

   seuratToGiotto_OLD(obj_use = NULL, ...)

Arguments
=========

=========== =========================
Argument    Description
=========== =========================
``obj_use`` Seurat object
``...``     additional params to pass
=========== =========================

Value
=====

Giotto object
