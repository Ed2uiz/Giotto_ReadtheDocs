
``showGiottoInstructions``
==============================

Show giotto instructions associated with giotto object

Description
-----------

Function to display all instructions from giotto object

Usage
-----

.. code-block:: r

   showGiottoInstructions(gobject)

Arguments
---------

.. list-table::
   :header-rows: 1

   * - Argument
     - Description
   * - ``gobject``
     - giotto object


Value
-----

named vector with giotto instructions
