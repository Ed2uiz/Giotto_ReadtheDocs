
``showGrids``
=================

showGrids

Description
-----------

Prints the available spatial grids that are attached to the Giotto object

Usage
-----

.. code-block:: r

   showGrids(gobject, verbose = TRUE)

Arguments
---------

.. list-table::
   :header-rows: 1

   * - Argument
     - Description
   * - ``gobject``
     - a giotto object
   * - ``verbose``
     - verbosity of function#'


Value
-----

vector
