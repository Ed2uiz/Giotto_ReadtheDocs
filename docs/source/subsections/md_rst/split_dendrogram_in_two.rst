split_dendrogram_in_two
-----------------------

Description
~~~~~~~~~~~

Merge selected clusters based on pairwise correlation scores and size of
cluster.

Usage
~~~~~

::

   split_dendrogram_in_two(dend)

Arguments
~~~~~~~~~

+-----------------------------------+-----------------------------------+
| ``dend``                          | dendrogram object                 |
+-----------------------------------+-----------------------------------+

Value
~~~~~

list of two dendrograms and height of node
