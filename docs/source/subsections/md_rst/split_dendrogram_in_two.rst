
``split_dendrogram_in_two``
===============================

split_dendrogram_in_two

Description
-----------

Merge selected clusters based on pairwise correlation scores and size of cluster.

Usage
-----

.. code-block:: r

   split_dendrogram_in_two(dend)

Arguments
---------

.. list-table::
   :header-rows: 1

   * - Argument
     - Description
   * - ``dend``
     - dendrogram object


Value
-----

list of two dendrograms and height of node
