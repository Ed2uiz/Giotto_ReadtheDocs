Wrap message
------------

Description
~~~~~~~~~~~

Wrap message

Usage
~~~~~

::

   wrap_msg(..., sep = " ")

Arguments
~~~~~~~~~

+-----------------------------------+-----------------------------------+
| ``...``                           | additional strings and/or         |
|                                   | elements to pass to cat           |
+-----------------------------------+-----------------------------------+
| ``sep``                           | how to join elements of string    |
|                                   | (default is one space)            |
+-----------------------------------+-----------------------------------+
| ``strWidth``                      | externally set wrapping width.    |
|                                   | (default value of 100 is not      |
|                                   | effected)                         |
+-----------------------------------+-----------------------------------+
| ``keywords``                      | internal                          |
+-----------------------------------+-----------------------------------+
