Wrap message

Usage
=====

.. code:: r

   wrap_msg(..., collapse = "")

Arguments
=========

+-------------------------------+--------------------------------------+
| Argument                      | Description                          |
+===============================+======================================+
| ``...``                       | additional strings and/or elements   |
|                               | to pass to cat                       |
+-------------------------------+--------------------------------------+
| ``collapse``                  | how to join elements of string       |
|                               | (default is no space)                |
+-------------------------------+--------------------------------------+
