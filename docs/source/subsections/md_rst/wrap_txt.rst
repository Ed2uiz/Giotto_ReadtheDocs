``wrap_txt``
============

Wrap text

Description
-----------

Wrap text

Usage
-----

.. code:: r

   wrap_txt(..., sep = " ", strWidth = 100)

Arguments
---------

+-------------------------------+--------------------------------------+
| Argument                      | Description                          |
+===============================+======================================+
| ``...``                       | additional params to pass            |
+-------------------------------+--------------------------------------+
| ``sep``                       | how to join elements of string       |
|                               | (default is one space)               |
+-------------------------------+--------------------------------------+
| ``strWidth``                  | externally set wrapping width.       |
|                               | (default value of 100 is not         |
|                               | effected)                            |
+-------------------------------+--------------------------------------+
