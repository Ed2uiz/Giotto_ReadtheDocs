# `S3toS4spatNetObj`

Spatial Networks


## Description

convert S3 spatialNetworkObj to S4


## Usage

```r
S3toS4spatNetObj(object, spat_unit = NULL)
```


## Arguments

Argument      |Description
------------- |----------------
`object`     |     S3 spatNetworkObj
`spat_unit`     |     spatial unit metadata to append
