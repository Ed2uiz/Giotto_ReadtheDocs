# `fDataDT`

fDataDT


## Description

show gene metadata


## Usage

```r
fDataDT(gobject)
```


## Arguments

Argument      |Description
------------- |----------------
`gobject`     |     giotto object


## Value

data.table with gene metadata


## Examples

```r
data(mini_giotto_single_cell) # loads existing Giotto object
fDataDT(mini_giotto_single_cell)
```
