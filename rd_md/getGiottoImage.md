# `getGiottoImage`

getGiottoImage


## Description

get get a giotto image from a giotto object


## Usage

```r
getGiottoImage(gobject, image_name)
```


## Arguments

Argument      |Description
------------- |----------------
`gobject`     |     giotto object
`image_name`     |     name of giotto image [`showGiottoImageNames`](#showgiottoimagenames)


## Value

a giotto image
