# `pDataDT`

pDataDT


## Description

show cell metadata


## Usage

```r
pDataDT(gobject)
```


## Arguments

Argument      |Description
------------- |----------------
`gobject`     |     giotto object


## Value

data.table with cell metadata


## Examples

```r
data(mini_giotto_single_cell) # loads existing Giotto object
pDataDT(mini_giotto_single_cell)
```
