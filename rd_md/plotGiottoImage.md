# `plotGiottoImage`

plotGiottoImage


## Description

get plot a giotto image from a giotto object


## Usage

```r
plotGiottoImage(gobject, image_name)
```


## Arguments

Argument      |Description
------------- |----------------
`gobject`     |     giotto object
`image_name`     |     name of giotto image [`showGiottoImageNames`](#showgiottoimagenames)


## Value

plot
